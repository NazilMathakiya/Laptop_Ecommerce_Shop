## 💻 Laptop E-Commerce Website

A **Laptop E-Commerce Website** designed to allow users to browse various laptop products, view detailed specifications, and easily purchase laptops online.

The system provides a **simple and user-friendly shopping experience**, including features like product search, cart management, and customer reviews.


## 🎮 Features

- 👤 User Registration and Login
- 🔍 Search Laptop Products
- 💻 View Laptop Details
- 🛒 Add to Cart Functionality
- ⚡ Buy Now Option
- 💰 Total Price Calculation
- ⭐ Product Review and Feedback
- 👤 User Profile Management
- 🚪 Logout System


## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL(Database)
- XAMPP

## 🚀 Future Enhancements

- 💳 Online payment gateway integration
- 🔔 Order status notifications
- ❤️ Wishlist feature for saving products
- 🧾 Invoice generation after purchase
